# ESP32 4-Channel DC Motor Driver @18V Max

* Max input Voltage - 18V
* Reverse Polarity Protection
* 5V Power to VIN Pin on ESP32 Module (for Wireless Functionaility)
* **4 GPIO's pulled high via 10K Resistor**
* GPIO27
* GPIO14
* GPIO16
* GPIO4
